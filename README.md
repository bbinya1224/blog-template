# 오롯이 (Orot-i)

> **경험은 당신이, 표현은 오롯이가.**

대화를 통해 잊혀진 경험의 디테일을 되살리고, 당신만의 언어로 풍부하게 기록해주는 AI 기반 경험 증폭 도구입니다.

---

## 왜 오롯이인가

우리는 매일 수많은 경험을 합니다. 맛있는 음식, 좋았던 제품, 감명 깊은 책. 하지만 그 순간의 감정과 디테일은 시간이 지나면 흐려집니다.

오롯이는 **대화**를 통해 그 기억의 조각들을 하나씩 끌어냅니다. 뇌과학적으로 설계된 질문 흐름이 잊혀진 세부 기억을 되살리고, AI가 그것을 당신만의 언어로 풍부하게 표현해줍니다.

당신은 경험만 하세요. 표현은 오롯이가 합니다.

---

## 핵심 기능

- **문체 학습** — 네이버 블로그 RSS 분석으로 나만의 톤을 추출
- **장소 정보 자동 수집** — 카카오 로컬 검색 + Tavily로 전화번호, 주소, 카테고리 등 실제 정보 수집
- **대화형 기록** — 채팅하듯 이야기하면 세부 기억이 되살아남
- **경험 증폭** — Claude AI가 내 말투로 1500~2000자의 풍부한 글 생성
- **편집 & 보관** — 생성된 글을 실시간 수정하고 저장

---

## 사용 흐름

```
1. 카테고리 선택     맛집 / 제품 / 뷰티 / 독서
        ↓
2. 대화             오롯이가 경험에 대해 질문합니다
        ↓
3. 기억 증폭        대화 속에서 세부 기억이 되살아납니다
        ↓
4. 글 완성          사용자의 톤으로 풍부한 글이 생성됩니다
        ↓
5. 편집 & 보관      마음에 드는 부분을 수정하고 저장합니다
```

## 카테고리

| 카테고리 | 설명 | 상태 |
|---------|------|------|
| 맛집 | 맛있었던 그 곳의 이야기 | 운영 중 |
| 제품 | 내가 산 물건 이야기 | 준비 중 |
| 뷰티 | 나의 뷰티 경험 | 준비 중 |
| 독서 | 읽고 느낀 것들 | 준비 중 |
